== Description ==

Zeige internationalen Lesern Deinen Inhalt in ihrer Sprache mit dem Google Übersetzung Widget. 

= Features =
Arbeitet nur als Widget
Erkennt automatisch die Sprache des Nutzers.

Nach der Aktivierung wirst Du automatisch  auf die Widget Seite umgeleitet, damit Du sofort Deine Einstellungen vornehmen kannst.

Bei Hilfe oder Fragen zu diesem Plugin schau auch hier vor bei http://internet-pr-beratung.de/simple-google-translate-widget/

== Installation ==

1. Lade die Zip-Datei in das 'plugins' Verzeichnis hoch oder direkt unter WP Admin -> Plugins -> und suche nach Simple Google Translate.
2. Aktiviere das Plugin
3. Du wirst automatisch auf Deine Widgets Seite zum einstellen umgeleitet.
